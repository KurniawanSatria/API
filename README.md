# Cyber API

Cyber API adalah REST API serbaguna yang menyediakan berbagai fitur keren mulai dari AI, media downloader, search engine tools, utilitas umum, hingga fitur hiburan. Cocok buat integrasi ke bot WhatsApp, Telegram, web app, atau mobile app.

## 🔥 Fitur Utama

### 🧠 AI
- Chat dengan AI
- Image generation
- Text summarizer
- Translate otomatis

### 📥 Downloader
- YouTube MP3/MP4 Downloader
- TikTok Video (No Watermark)
- Instagram Media Downloader
- Facebook, Twitter, dsb

### 🔎 Search
- Google Search API
- YouTube Search
- Image Search
- Wikipedia Scraper

### 🛠 Tools
- Background Remover
- Image Upscaler
- Text-to-Speech
- QR Code Generator
- URL Shortener

### 🎮 Entertainment
- Random Anime/Meme
- Truth or Dare Generator
- Game Trivia
- Chatbot Personality Switcher

## 🚀 Cara Install & Run

```bash
# clone repo
git clone https://github.com/yourusername/cyber-api
cd cyber-api

# install dependencies
npm install # atau pip install -r requirements.txt (tergantung bahasa)

# run server
npm start # atau python app.py
