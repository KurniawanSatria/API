# Cyber API

Cyber API adalah REST API serbaguna yang menyediakan berbagai fitur keren mulai dari AI, media downloader, search engine tools, utilitas umum, hingga fitur hiburan. Cocok buat integrasi ke bot WhatsApp, Telegram, web app, atau mobile app.

## 🔥 Fitur Utama

### 🧠 AI
- Chat dengan AI
- Image Enhancer
- Image Removebg

### 📥 Downloader
- YouTube MP3/MP4 Downloader
- Spotify Downloader
- Soundcloud Downloader

### 🔎 Search
- YouTube Search
- Spotify Search
- Soundcloud Search
- Lyrics Search

### 🛠 Tools
- Day Off Checker
- QR Code Generator
- Spotify Daily Chart

### 🎮 Entertainment
- Random Anime Picture
- Anime Quote
- Joke
- Meme
- Rizz
- Fun Facts
